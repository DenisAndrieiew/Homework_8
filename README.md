Задача была в создании приложения для учета производителей и продуктов. 
Каждый продукт соотносится с одним производителем, продуктов у производителя может 
быть неограниченное количество. Пользователь может просматривать продукты и производителей,
пользователь с правами администратора может создавать, изменять и удалять продукты и производителей,
а так-же смотреть информацию о пользователях и изменять их данные и роль.

Использованные технологии: Java, Spring (MVC, Security, Data, Spring Boot), PostreSQL,
Gradle, git, Tomcat.

Для запуска: в PGAdmin последовательно запустить скрипты из файлов ddl.sql затем dml.sql.
После этого запустить приложение и создать пользователя.
При необходимости получения прав администратора - первому пользователю скопировать в поле
role_id uuid роли администратора. В дальнейшем назначать новых администраторов можно при
использовании приложения существующим администратором.
